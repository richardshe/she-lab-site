# Introduction
The discovery that genetic information is encoded in the linear sequence of four nucleotide bases in deoxyribonucleic acid (DNA) posed the fundamental problem of how this sequence dictates the specific amino-acid sequences of proteins. The rules governing this translation constitute the genetic code. While DNA replication is well understood, the general nature of the genetic code has remained a subject of intense speculation.

Key questions include the number of bases required to specify an amino-acid (the coding ratio), whether the sequence is overlapping or non-overlapping, and how the reading frame is established along a continuous nucleotide strand. Recent studies on tobacco mosaic virus mutants and human haemoglobin variants suggest the code is non-overlapping, since single mutations typically alter only one amino-acid. If the code is non-overlapping, a mechanism must ensure the bases are read in the correct groupings. In an overlapping code, special "comma" sequences might separate words, but this restricts the allowed sequences. Alternatively, the sequence could be read sequentially from a fixed starting point. Here, we report genetic experiments on the rII region of bacteriophage T4 which provide compelling evidence that the genetic code is non-overlapping, is read from a fixed starting point in groups of three (or a multiple thereof), and is most likely degenerate, meaning multiple triplets can code for the same amino-acid.
