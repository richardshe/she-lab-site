# Abstract

A fundamental question in neuroscience is whether a defined population of neurons, or a memory engram, is sufficient to elicit the recall of a specific memory. While studies have shown that specific cell populations are necessary for memory expression, mimicking memory recall by directly activating these cells has remained elusive. The dentate gyrus (DG) of the hippocampus is known to be critical for contextual memory formation and recruits sparse, distinct populations of neurons for different environments. Here, we demonstrate that optogenetic activation of a specific DG neural population is sufficient to induce the behavioral recall of a fear memory. We developed a viral and transgenic strategy using c-fos-tTA mice to selectively label DG neurons that were naturally active during contextual fear conditioning with the light-activated cation channel Channelrhodopsin-2 (ChR2). Subsequent optical stimulation of these specific, learning-activated cells in a completely different, unconditioned context elicited robust freezing behavior, indicating successful fear memory recall. Crucially, this light-induced freezing was not observed when we optically stimulated DG cells labeled in a neutral, unconditioned context, nor did it occur in control mice expressing only a fluorescent protein. These results demonstrate that a specific contextual memory is encoded within a discrete ensemble of hippocampal DG neurons and that direct, artificial reactivation of this engram is sufficient to drive the associated learned behavior. Our findings provide direct causal evidence for the existence of cellular memory engrams and offer a powerful approach to map the complex neural circuits underlying memory formation and retrieval.
