A specific memory is thought to be encoded by a sparse population of neurons whose coordinated activity defines a cellular memory engram. Selective ablation or inhibition of such populations has been shown to disrupt memory expression, indicating that these cells are necessary for memory recall. However, the demonstration that a defined neuronal population constitutes the cellular basis of a particular memory requires a mimicry experiment, in which direct reactivation of that population is itself sufficient to elicit the associated behavioural output. Here we test this prediction in the dentate gyrus (DG) of the hippocampus, a region implicated in the formation of contextual memory representations. We used c-fos-tTA transgenic mice in combination with an AAV9-TRE-ChR2-EYFP virus to label, in an activity-dependent and doxycycline-regulated manner, the sparse population of DG granule cells that were active during contextual fear conditioning. In vivo recordings confirmed that these cells could be reliably driven by blue light pulses delivered through an implanted optical fibre, and that optical stimulation also engaged downstream CA3 neurons. When mice that had been fear conditioned in one context were placed in a distinct, neutral context, optical reactivation of the labelled DG ensemble was sufficient to evoke freezing behaviour, whereas mice receiving no shock during training, mice expressing EYFP alone, or mice in which the labelled cells were active in an unrelated context did not freeze. Bilateral stimulation produced freezing approaching that elicited by the conditioned context itself. These findings demonstrate that a sparse population of DG cells active during learning forms a contextual memory engram whose direct reactivation is sufficient for fear memory recall.
